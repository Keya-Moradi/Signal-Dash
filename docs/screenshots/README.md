# Screenshots

This folder contains screenshots for the main README.

## Required Screenshots

To complete the documentation, add the following screenshots:

1. **dashboard.png** - Main experiment dashboard showing list of experiments with status badges
2. **experiment-detail.png** - Experiment detail view showing variants, stats, and visualization
3. **readout.png** - Readout page with AI-generated summary and decision form

## How to Capture

1. Start the app: `npm run dev`
2. Navigate to http://localhost:3000
3. Capture screenshots at reasonable browser width (1200-1400px recommended)
4. Save as PNG files in this directory

## Used In

These images are embedded in the main README.md Screenshots section.
